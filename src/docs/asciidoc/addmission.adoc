== 개요

참가자들의 상태를 조회하고 관련 정보를 관리하는 API입니다. +
참가자 정보를 조회하여 현재 상태(합격 여부, 등록 여부 등)를 확인할 수 있습니다.

---

== 참가자 상태 조회

이름과 이메일을 통해 참가자의 합격 여부와 상태를 조회합니다.

=== 상태 조회 (200 OK)

참가자 조회가 성공했을 경우, 해당 참가자의 상태 정보를 반환합니다.
