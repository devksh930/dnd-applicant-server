== 개요

사용자 인증 및 권한 관리를 위한 API입니다. +
로그인/로그아웃을 통해 세션과 토큰(JWT)을 관리합니다.

---

== 로그인

=== 1. 로그인 성공 (200 OK)

이메일과 비밀번호를 사용하여 로그인을 시도합니다. +
성공 시 `Authorization` 헤더와 응답 본문에 액세스 토큰이 포함됩니다.
