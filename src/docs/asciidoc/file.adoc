== 개요

프로젝트 진행 중 필요한 파일을 서버에 업로드하는 API입니다. +
단일 파일 업로드와 다중 파일 업로드를 지원하며, `multipart/form-data` 형식을 사용합니다.

[NOTE]
.Content-Type
====
파일 업로드 요청 시 `Content-Type`은 반드시 `multipart/form-data`여야 합니다.
====

---
