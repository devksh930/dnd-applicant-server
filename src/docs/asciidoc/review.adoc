[NOTE]
====
요청 본문 중 `urlLinks[].type` 필드는 Enum(`UrlType`)입니다. Spring REST Docs 표에서는 JSON 타입 기준으로 `STRING`으로 표시되지만, 실제로는 아래 Enum 값만 허용됩니다.

- `LINKEDIN`(링크드인)
- `GITHUB`(깃허브)
- `EMAIL`(이메일)
- `TISTORY`(티스토리)
- `VELOG`(벨로그)
- `MEDIUM`(미디엄)
- `BEHANCE`(비헨스)
- `INSTAGRAM`(인스타그램)
- `YOUTUBE`(유튜브)
====
