== 개요

프로젝트 제출 및 관리와 관련된 API입니다. +
발급된 링크 키(`linkKey`)를 통해 프로젝트 정보를 제출하거나 수정할 수 있습니다.

주요 기능은 다음과 같습니다:

* **최초 제출**: 링크를 통해 처음 프로젝트를 등록합니다.
* **재제출**: 이미 등록된 프로젝트 정보를 수정합니다.
* **유효성 검증**: 링크의 유효 기간이 만료되었는지 확인합니다.

---

== 프로젝트 제출

프로젝트 정보를 제출하는 API입니다. +
동일한 링크 키(`linkKey`)를 사용하여 최초 제출과 재제출(수정)을 모두 처리할 수 있습니다.

[NOTE]
.Link Key란?
====
각 팀에게 부여된 고유한 UUID입니다. 이 키를 통해 팀을 식별하고 프로젝트를 제출할 권한을 얻습니다.
====

=== 1. 최초 제출 (201 Created)

팀이 프로젝트 정보를 처음으로 제출하는 경우입니다. +
성공 시 `201 Created` 상태 코드와 함께 리소스의 위치(`Location`)가 헤더로 반환됩니다.

[TIP]
.UrlType (링크 타입) 상세
====
`urlLinks[].type` 필드는 아래의 정해진 값(Enum)만 허용됩니다.

* `GITHUB` : 깃허브 저장소
* `YOUTUBE` : 유튜브 영상
* `LINKEDIN` : 링크드인 프로필
* `EMAIL` : 이메일 주소
* `TISTORY` : 티스토리 블로그
* `VELOG` : 벨로그
* `MEDIUM` : 미디엄
* `BEHANCE` : 비헨스 포트폴리오
* `INSTAGRAM` : 인스타그램
