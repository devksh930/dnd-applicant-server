== API 목록

각 항목을 클릭하면 상세 문서로 이동합니다.

* link:addmission.html[**참가(Admission)**]
** 참가자 상태 조회 및 관리
* link:auth.html[**인증(Auth)**]
** 로그인 및 토큰 발급
* link:review.html[**프로젝트(Review)**]
** 프로젝트 제출, 재제출, 링크 검증, 멤버 리뷰 목록 조회
* link:link.html[**링크(Link)**]
** 링크 유효성 및 타입 조회
* link:file.html[**파일(File)**]
** 단일 및 다중 파일 업로드
* link:error.html[**에러(Error)**]
** 전역 에러 코드 및 예외 처리 응답 구조
