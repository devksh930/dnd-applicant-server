== 개요

발급된 링크의 유효성을 검증하고 타입(프로젝트 제출용 등)을 확인하는 API입니다. +
링크 키(UUID)를 통해 해당 링크가 유효한지, 만료되지는 않았는지 미리 확인할 수 있습니다.

---
